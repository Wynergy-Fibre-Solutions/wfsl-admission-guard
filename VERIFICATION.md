\# Verification



This repository enforces admission control for declared system surfaces.



\## Verification intent



Verification confirms that:



\- admission rules are explicit and deterministic

\- undeclared routes or behaviours are denied by default

\- changes to admission rules are auditable



\## Verification scope



This verification does not assert:

\- security guarantees

\- compliance guarantees

\- runtime correctness beyond declared admission rules



\## Minimum verification requirements



WFSL verification requires:

1\. Apache License 2.0 present

2\. Governance artefacts present

3\. Deterministic verification runner passes

4\. Evidence artefact is committed
